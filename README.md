# オセロ（Othello）ゲーム

Python と Tkinter を用いて作成した、自作のオセロ（リバーシ）ゲームです。
プレイヤーは白石、コンピュータは黒石を担当し、モンテカルロ法や単純ヒューリスティックによる思考アルゴリズムを搭載しています。

## 📂 ファイル構成

othello_64.py
プレイヤー vs コンピュータの GUI オセロゲーム。コンピュータはモンテカルロ法で手を選択します。

othello_AI.py
AI 同士が自動で対戦し、勝率や対戦数をコンソールに出力します。AI には以下の 2 種類の手法を実装しています：

AI_1: 相手の置ける手数を最小化する戦略

AI_2: モンテカルロ法による勝率最大化戦略

## 🚀 必要環境

Python 3.x

Tkinter（Python 標準ライブラリ）

実行環境は Windows / macOS / Linux に対応（Tkinter が動作する必要あり）

## 🔧 実行方法
GUI 版（othello_64.py）
python othello_64.py


マウスクリックで石を配置できます。

配置可能なマスは黄色で表示されます。

ゲーム終了時に勝敗がダイアログで表示されます。

AI 対戦版（othello_AI.py）
python othello_AI.py


AI 同士の対戦を自動で繰り返します。

コンソールに勝率・勝数・引き分け数・総対戦数が出力されます。

## 🧠 実装されている AI ロジック

ランダム手（デバッグ用）

ヒューリスティック手: 次の相手の置ける手数を最小化する戦略

モンテカルロ法: 複数回のランダムシミュレーションにより勝率の高い手を選択
