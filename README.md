# オセロゲーム

Python（Flask）と JavaScript で作成した、ブラウザで遊べるオセロ（リバーシ）ゲームです。  
プレイヤー（白）vs AI（黒）の 1 対 1 形式で、ゲーム前に AI の難易度を選択できます。

---

## ファイル構成

```
othello/
├── app.py               # Flask バックエンド（ゲームロジック・API）
├── requirements.txt     # 依存パッケージ
└── templates/
    └── index.html       # フロントエンド（UI・ゲーム描画）
```

---

## 機能

- **難易度選択**：ゲーム開始前に 簡単 / 普通 / 難しい の 3 段階から選択
- **先手後手**：ランダムに決定（AI が先手の場合は自動で着手）
- **置けるマスの表示**：自分のターン中、有効なマスをドットでハイライト
- **最後の手のハイライト**：自分の手（金色）・AI の手（赤色）を盤面上で強調
- **石のアニメーション**：新規配置・ひっくり返った石にポップインエフェクト
- **パス通知**：どちらかがパスになる場合はトースト表示で自動進行
- **勝敗画面**：ゲーム終了時に Win（赤）/ Lose（青）/ Draw とスコアを表示

---

## AI ロジック

| 難易度 | 手法 | 概要 |
|---|---|---|
| 簡単 | ランダム | 置けるマスからランダムに選択 |
| 普通 | ヒューリスティック | 自分が置いた後に相手が打てるマス数を最小化する 1 手読み |
| 難しい | モンテカルロ法 | 各候補手から終局までランダムシミュレーション（10 回）し、最も勝率の高い手を選択 |

## 必要環境

- Python 3.x
- Flask 3.x

---

## セットアップ・実行方法

**1. 依存パッケージのインストール**

```bash
pip install -r requirements.txt
```

**2. サーバーの起動**

```bash
python app.py
```

**3. ブラウザでアクセス**

```
http://127.0.0.1:5000
```
