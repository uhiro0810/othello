# オセロ（Othello）ゲーム

Python と Tkinter を用いて作成した、自作のオセロ（リバーシ）ゲームです。

## ファイル構成

| ファイル | 概要 |
|---|---|
| `othello_64.py` | プレイヤー vs コンピュータの GUI 対戦版 |
| `othello_AI.py` | AI 同士が自動で連続対戦し、統計をコンソールに出力する計測版 |

---

### othello_64.py（GUI 対戦版）

- **プレイヤー**：白石　**コンピュータ**：黒石
- 先手後手はゲーム開始時にランダムに決定されます
- コンピュータの手は **モンテカルロ法**（各候補手につき 10 回ランダムシミュレーション）で選択されます
- 配置可能なマスは **黄色（gold）** でハイライト表示されます
- パスが必要な場合や、ゲーム終了時の勝敗はダイアログで通知されます

### othello_AI.py（AI 自動対戦版）

- **白（WHITE）**：ランダムに手を選択
- **黒（BLACK）**：**AI_2（モンテカルロ法、10 ループ）** で手を選択
- `AI_1`（ヒューリスティック：相手の置けるマス数を最小化）はコード内にあり、コメントアウトで切り替え可能です
- ダイアログは表示せず、1 ゲームごとに以下をコンソールへ出力します：

  ```
  白：<白勝利数>  黒：<黒勝利数>  引き分け：<引き分け数>  総プレイ数：<総数>  AIの勝率<黒勝率%>
  ```

---

## 実装されている AI ロジック

| 名称 | 概要 |
|---|---|
| ランダム | 置けるマスからランダムに選択（`othello_AI.py` の白側で使用） |
| AI_1（ヒューリスティック） | 自分が石を置いた後に相手が置けるマスの数を最小化する貪欲法 |
| AI_2（モンテカルロ法） | 各候補手から終局までランダムにシミュレーションし、最も勝率の高い手を選択 |

---

## 必要環境

- Python 3.x
- Tkinter（Python 標準ライブラリ、追加インストール不要）
- Windows / macOS / Linux（Tkinter が動作する環境であれば可）

## 実行方法

**GUI 対戦版**（プレイヤー vs コンピュータ）

```bash
python othello_64.py
```

**AI 自動対戦版**（連続対戦・統計計測）

```bash
python othello_AI.py
```

> `othello_AI.py` は停止するまで対戦を繰り返します。終了は `Ctrl+C` またはウィンドウを閉じてください。
